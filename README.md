# Face Restoration Analytics Dashboard

An interactive Streamlit dashboard developed as part of an academic project on **AI-Based Face Restoration and 3D Reconstruction**. The dashboard evaluates face restoration performance using image quality metrics, statistical analysis, and machine learning.

---

## Overview

This dashboard provides an interface to analyze and compare the performance of AI-based face restoration techniques using metrics such as:

- PSNR (Peak Signal-to-Noise Ratio)
- SSIM (Structural Similarity Index)
- NIQE (Natural Image Quality Evaluator)

The application combines exploratory data analysis, statistical testing, and predictive modeling into a single interactive dashboard.

---

## Features

- Interactive analytics dashboard using Streamlit
- Exploratory Data Analysis (EDA)
- Distribution and correlation analysis
- One-Way ANOVA statistical testing
- Linear Regression model for PSNR prediction
- Restoration Insights
- Image comparison module
- Degradation-wise performance analysis
